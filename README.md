# Speech to Text sync

Sync an audio file with text counterpart 

## Install

```
git clone git@github.com:fijiwebdesign/speech2text-sync.git
cd speech2text-sync
npm install
npm start
```

## Browser support

`AudioContext` - https://caniuse.com/?search=AudioContext
`WebAudio` - https://caniuse.com/?search=WebAudio
